## 📌 About

**GreenWallet** is a web application that helps users track their carbon footprint, monitor eco-friendly spending, and make more sustainable lifestyle choices. The platform combines financial tracking with environmental impact awareness to encourage greener daily decisions.

> 🌱 Frontend built with vanilla HTML, CSS, and JavaScript. Backend coming soon.

## ✨ Features

### 🏠 Landing Page
- Hero section with platform introduction
- Feature highlights and benefits
- Call-to-action for signup/login

### 🔐 Authentication
- Clean login and registration UI
- Form validation
- Session-aware navbar (shows user info when logged in)

### 📊 Dashboard
- Personal carbon footprint overview
- Eco spending tracker
- Progress indicators and stats
- Interactive charts and data visualization

### 🧮 Carbon Calculator
- Calculate carbon emissions from daily activities
- Transport, food, energy usage inputs
- Visual results with reduction suggestions

### 👤 Profile Page
- User profile management
- Eco score and achievements
- Personal sustainability goals

### ℹ️ About & Features Pages
- Platform mission and vision
- Detailed feature showcase
- Team and project information

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Edge)
- A local server (Live Server extension in VS Code recommended)

### Run Locally

**Option 1 — VS Code Live Server (recommended)**
1. Clone the repo:
```bash
git clone https://github.com/mehmoona-chand/greenwallet-frontend.git
```
2. Open the folder in VS Code
3. Right-click `index.html` → **Open with Live Server**

**Option 2 — Direct browser**
1. Clone the repo
2. Open `index.html` directly in your browser
